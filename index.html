The Challenge

200,000+ sq ft with no visual navigation system

The client operated a sprawling multi-floor gaming and entertainment complex spanning casino floors, restaurants, event spaces, and parking structures. Guest navigation relied entirely on staff — no cohesive wayfinding system existed across the property. The ask: build one from scratch that feels native to the brand.

Deliverables

14 sign types. One master system.

🏛
Entry Totems

Exterior freestanding, 12ft illuminated panels. Color-matched to brand palette.

🗺
Directory Panels

Wall-mounted at decision points. Die-cut acrylic with backlighting specs.

➡️
Directional Blades

Double-sided hanging signs at corridor intersections. Vinyl on aluminum composite.

🎰
Zone IDs

Large-format gaming section headers. Color system per zone, illuminated channel specs.

🅿️
Parking Structure

Exterior wayfinding, ADA routing, level identification, entrance/exit.

📋
Production Spec Guide

Full doc: substrates, vinyl types, color codes, hardware, installation notes.

Process

Site audit → system design → fabrication-ready output

Walked all 4 floors to identify 38 critical wayfinding nodes. Built a sub-palette aligned to brand standards while meeting ADA contrast requirements. Designed the full system in Illustrator with linked master components — changes propagate across all 14 sign types instantly. Specified 3M IJ180 for flat surfaces, Avery 1005EZ for curved substrates, with full Pantone-to-CMYK-to-vinyl color matching. Delivered layered print-ready PDFs with bleed, crop marks, cut paths, and per-sign production notes.
